üöÄ STEP-BY-STEP DEPLOYMENT PROCESS

1

Prepare PostgreSQL Database

Action: Create the required database and user

Instructions:

  1. Open Command Prompt as Administrator (Win + R ‚Üí cmd ‚Üí Ctrl+Shift+Enter)
  2. Connect to PostgreSQL:
    psql -U postgres
  3. If prompted for password, enter your PostgreSQL password
  4. Run these SQL commands in the psql shell:
    CREATE DATABASE "Backtest_Pulse"; CREATE USER postgres WITH PASSWORD 'Alliswell@28'; GRANT ALL PRIVILEGES ON DATABASE "Backtest_Pulse" TO postgres; ALTER USER postgres CREATEDB; \q
Expected Result: Database created successfully

üîß Troubleshooting:

  • "user already exists" error: Skip CREATE USER line, just run GRANT
  • Connection fails: Check PostgreSQL service is running (services.msc)
2

Set Up Project Directory

Action: Navigate to your project folder

Instructions:

cd C:\path\to\your\New_BackTest_Pulse\folder
Note: Replace C:\path\to\your\New_BackTest_Pulse\folder with your actual folder path
Expected Result: Command prompt shows your project directory path
3

Create Python Virtual Environment

Action: Set up isolated Python environment

Instructions:

# Create virtual environment python -m venv venv # Activate virtual environment venv\Scripts\activate
Expected Result:
  • (venv) appears at the start of your command prompt
  • Command prompt shows: (venv) C:\path\to\project>
Important: Keep this command prompt window open for all subsequent steps
4

Install Python Dependencies

Action: Install required Python packages

Instructions:

pip install -r requirements.txt
Expected Result:
  • Multiple packages install (takes 1-2 minutes)
  • No error messages
  • Final output shows "Successfully installed..."

üîß Troubleshooting:

  • "pip not recognized": Run python -m pip install -r requirements.txt
  • Network errors: Check internet connection
5

Configure Environment Variables

Action: Set database connection parameters

Instructions:

In the SAME command prompt, run:

set PGHOST=localhost set PGPORT=5432 set PGDATABASE=Backtest_Pulse set PGUSER=postgres set PGPASSWORD=Alliswell@28
Expected Result: No output (variables are set silently)

Verification:

Run echo %PGDATABASE% - should show Backtest_Pulse

üîß TROUBLESHOOTING GUIDE

Issue: "psql: command not found"

Solution:

  • Add PostgreSQL bin folder to PATH
  • Or use full path: "C:\Program Files\PostgreSQL\14\bin\psql.exe"

Issue: "python: command not found"

Solution:

  • Use py instead of python
  • Or use full path: "C:\Python38\python.exe"

Issue: Database connection fails

Solution:

# Test connection psql -U postgres -d Backtest_Pulse -c "SELECT 1;" # Check PostgreSQL service services.msc # Find "postgresql-x64-14" and ensure it's running

Issue: Port 5000 already in use

Solution:

# Find process using port netstat -ano | findstr :5000 # Kill the process taskkill /PID <PID_NUMBER> /F

Issue: Virtual environment not activating

Solution:

# Use full path C:\path\to\project\venv\Scripts\activate.bat # Or run as administrator

Issue: Pip install fails

Solution:

# Upgrade pip first python -m pip install --upgrade pip # Then install requirements pip install -r requirements.txt
